# XML Foundations

If you work primarily with JSON, XML can feel verbose at first. This section covers the essentials you need to be productive — no more, no less — and draws direct comparisons to what you already know.

## What's Here

- **[XML vs JSON](xml-vs-json.md)** — A side-by-side comparison for developers who think in JSON. What XML does differently, and why.
- **[Document Structure](document-structure.md)** — Elements, attributes, namespaces, and the document tree. The building blocks everything else depends on.
- **[The XDM](xdm.md)** — The XQuery and XPath Data Model. How XML tools see your documents — and why it matters for writing effective queries and transforms.
- **[Namespaces](namespaces.md)** — The most confusing part of XML, explained practically. When you need them, when you don't, and how to handle them in .NET.
- **[Working with JSON](working-with-json.md)** — Parse, query, transform, and output JSON using XPath, XSLT, and XQuery. The bridge between formats.
- **[Unicode and .NET Requirements](unicode-and-dotnet.md)** — ICU globalization, Unicode normalization, collation, and platform setup for PhoenixmlDb.
